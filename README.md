# Speed Youtube

Simple Youtube Extension for chrome to speed up videos or slow down by up to 10X.
Set the increment/decrement steps, then use Fast/Slow/Reset as required.

## Installation

1. Save this repository in a folder anywhere on your system.
2. Open Chrome > Manage Extensions > Developer mode On
3. Load unpacked > pick the folder
4. Activate it > pin it to extension bar
5. You should see the buttons activated on any youtube page
